# 🎣 Phishing & Ingénierie sociale

---

# 📌 Définition

## 🎣 Phishing

Le **phishing** est une attaque visant à :

- tromper une victime
- pour récupérer des informations sensibles

👉 exemple :

- mots de passe
- données bancaires

---

## 🧠 Ingénierie sociale

Technique consistant à :

- manipuler une personne
- pour obtenir un accès ou une information

👉 attaque basée sur le facteur humain

---

# 🧠 Principe

```
Attaquant → Manipulation → Victime → Information
```

# 📦 Exemples concrets

## Email frauduleux

- "Votre compte est bloqué"
- "Cliquez ici"

---

## Faux support IT

- appel téléphonique
- demande mot de passe

---

## USB piégée

- clé laissée volontairement
