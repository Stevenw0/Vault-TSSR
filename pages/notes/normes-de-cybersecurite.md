# 🛡️ Normes de cybersécurité

---

# 📌 Définition

Les **normes de cybersécurité** sont des référentiels permettant de :

- protéger les systèmes d’information
- gérer les risques
- standardiser les bonnes pratiques

---

# 🧠 Objectif

```
Sécuriser → Standardiser → Améliorer
```

---

# 🎯 Rôle

| Fonction | Description |
|---|---|
| sécurité | protection SI |
| conformité | respect règles |
| audit | contrôle |
| gouvernance | organisation |

---

# 📊 Principales normes

| Norme | Description |
|---|---|
| ISO 27001 | management sécurité SI |
| ISO 27002 | bonnes pratiques |
| NIST | framework sécurité |
| CIS Controls | mesures concrètes |
| PCI-DSS | paiement |
| RGPD | données personnelles |

---

# 🔐 ISO 27001

## Définition

Norme internationale pour :

- mettre en place un SMSI (Système de Management de la Sécurité de l’Information)

---

## Objectifs

| Objectif | Description |
|---|---|
| gestion risques | analyse |
| sécurité | organisation |
| amélioration | continue |

---
