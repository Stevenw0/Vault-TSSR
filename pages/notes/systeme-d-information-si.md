# 🏢 Système d’Information (SI) — Vision Cybersécurité

---

# 📌 Définition

Un **système d’information (SI)** est l’ensemble des éléments permettant de :

- collecter
- stocker
- traiter
- diffuser

l’information d’une organisation

---

# 🧠 Vision globale

```
Utilisateurs → SI → Données → Services
```

---

# 🎯 Objectif du SI

| Objectif | Description |
|---|---|
| support métier | activité |
| gestion données | stockage |
| communication | interne/externe |
| automatisation | processus |

---

# 🧩 Composants du SI

| Élément | Description |
|---|---|
| utilisateurs | humains |
| matériel | serveurs, PC |
| logiciels | apps |
| données | informations |
| réseau | communication |

---

# 🏗️ Architecture

```
Users
 ↓
Applications
 ↓
Serveurs
 ↓
Base de données
 ↓
Infrastructure réseau
```

---

# 🔐 SI en cybersécurité

Le SI devient une **surface d’attaque** :

👉 tout ce qui est connecté peut être ciblé

---

# 🎯 Objectifs cybersécurité

| Objectif | Description |
|---|---|
| confidentialité | protéger données |
| intégrité | éviter modification |
| disponibilité | accès continu |

---

👉 appelé triade **CIA**
