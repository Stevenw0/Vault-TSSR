# ⚙️ Méthode Agile & ITIL — Complémentarité

---

# 📌 Définition

## Agile

Méthode basée sur :

- flexibilité
- itérations rapides
- adaptation continue

---

## ITIL

Framework de gestion des services IT basé sur :

- processus
- stabilité
- qualité de service

---

# 🧠 Vision globale

👉 Agile et ITIL ne s’opposent pas :

```
Agile → développement rapide
ITIL → gestion stable des services
```

---

# 🎯 Objectifs

| Méthode | Objectif |
|---|---|
| Agile | livrer vite |
| ITIL | garantir qualité |

---

# 📊 Comparatif

| Critère | Agile | ITIL |
|---|---|---|
| approche | flexible | structurée |
| rythme | rapide | contrôlé |
| focus | développement | service |
| changement | fréquent | maîtrisé |

---

# 🧩 Complémentarité

| Agile | ITIL |
|---|---|
| développement | exploitation |
| sprint | change management |
| CI/CD | release management |
| feedback | amélioration continue |

# 🎯 Conclusion

👉 Agile permet :

- d’aller vite
