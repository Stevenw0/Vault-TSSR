# 💣 Ransomware — Attaque par rançongiciel

---

# 📌 Définition

Un **ransomware** est un logiciel malveillant qui :

- chiffre les données
- bloque l’accès au système
- demande une rançon pour les récupérer

---

# 🧠 Principe

```
Infection → Chiffrement → Blocage → Demande de rançon
```

---

# 🎯 Objectif

| Objectif | Description |
|---|---|
| extorsion | paiement |
| blocage | activité |
| pression | victime |
| gain financier | attaquant |

---

# 📊 Types de ransomware

| Type | Description |
|---|---|
| crypto-ransomware | chiffre fichiers |
| locker | bloque système |
| double extorsion | chiffre + vole données |
| wiper (faux) | détruit données |

---

# ⚙️ Fonctionnement

## Étapes

1. infection (phishing, vulnérabilité…)
2. propagation réseau
3. chiffrement fichiers
4. affichage rançon
5. paiement (souvent crypto)

---

# 📦 Exemple

```
Vos fichiers sont chiffrés !
Payez 1 BTC pour récupérer vos données.
```

---

---

# 🔐 Règle 3-2-1 (backup)

| Règle | Description |
|---|---|
| 3 copies | données |
| 2 supports | différents |
| 1 hors ligne | offline |

---

# ⚠️ Bonnes pratiques

> [!warning]

- ne jamais payer (si possible)
- sauvegarder régulièrement
- désactiver RDP exposé
- utiliser MFA
- surveiller logs

---

# 🧠 Réaction en cas d’attaque

## Étapes

1. isoler machine
2. couper réseau
3. analyser incident
4. restaurer sauvegarde
5. notifier
