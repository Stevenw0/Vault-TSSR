# 🌐 Classification des réseaux

Les réseaux informatiques peuvent être classés selon plusieurs critères.

---

# 1️⃣ Classification selon la mobilité

## Réseau filaire

Réseau utilisant des **câbles physiques**.

Exemples :

- Ethernet
- fibre optique
- câble cuivre

Avantages :

- stabilité
- débit élevé
- faible latence

---

## Réseau sans fil

Réseau utilisant des **ondes radio**.

Exemples :

- Wi-Fi
- Bluetooth
- 4G / 5G

Avantages :

- mobilité
- facilité de déploiement

Inconvénients :

- interférences
- sécurité plus complexe

---

# 2️⃣ Classification selon la topologie

## Topologie en étoile

Tous les équipements sont reliés à un **équipement central**.

Exemple :

```
     PC
      |
PC — Switch — PC
      |
     PC
```

Avantages :

- facile à gérer
- panne isolée

---

## Topologie en arbre

Structure **hiérarchique** composée de plusieurs étoiles.

```
        Core
       /   \
   Switch  Switch
   /   \    /   \
  PC   PC  PC   PC
```

---

## Topologie en anneau

Chaque équipement est relié à **deux voisins**.

```
PC — PC — PC
 |        |
 PC — PC —
```

---

## Topologie maillée

Chaque équipement peut être connecté à plusieurs autres.

Très utilisé dans :

- Internet
- réseaux opérateurs.

---

| Type | Signification | Exemple |
|---|---|---|
