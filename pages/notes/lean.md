# ⚙️ Lean — Amélioration continue & optimisation

---

# 📌 Définition

Le **Lean** est une méthode de gestion visant à :

- éliminer les gaspillages
- optimiser les processus
- améliorer la valeur pour le client

👉 Origine : industrie (Toyota)

---

# 🧠 Principe

```
Max valeur client
↓
Min gaspillage
↓
Amélioration continue
```

---

# 🎯 Objectifs

| Objectif | Description |
|---|---|
| efficacité | moins de pertes |
| qualité | amélioration |
| rapidité | processus fluide |
| coût | réduction |

---

# 📊 Les 7 gaspillages (Muda)

| Type | Description |
|---|---|
| surproduction | produire trop |
| attente | temps perdu |
| transport | déplacements inutiles |
| sur-processus | complexité |
| stock | trop de stockage |
| mouvement | actions inutiles |
| défaut | erreurs |

---

# 🧩 Concepts clés

| Concept | Description |
|---|---|
| Kaizen | amélioration continue |
| Just-In-Time | produire au bon moment |
| Value Stream | flux de valeur |
| Standardisation | processus clair |

---

# ⚙️ Méthode Lean

## Cycle

```
Observer → Analyser → Améliorer → Standardiser
```
