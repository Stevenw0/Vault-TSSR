# 🖥️ Desktop Shell — Environnements & Shells graphiques Linux

---

# 📌 Définition

Un **desktop shell** est la couche graphique qui :

- définit l’interface utilisateur (UI)
- gère les menus, barres, notifications
- permet d’interagir avec le système

👉 partie visible du Desktop Environment

---

# 🧠 Principe

```
Utilisateur → Desktop Shell → Compositor → Système
```

---

# 🎯 Objectifs

| Objectif | Description |
|---|---|
| ergonomie | interface |
| interaction | utilisateur |
| productivité | accès rapide |
| personnalisation | UI |
