# 🔐 Triade CIA — Fondements de la cybersécurité

---

# 📌 Définition

La **triade CIA** est le modèle de base de la sécurité de l’information :

```
C = Confidentialité
I = Intégrité
A = Disponibilité
```

👉 Elle définit les 3 objectifs essentiels de la cybersécurité

---

# 🧠 Principe

```
Donnée sécurisée = CIA respectée
```

---

# 🎯 Objectif global

| Objectif | Description |
|---|---|
| protéger | données |
| garantir | fiabilité |
| assurer | accès |

---

# 🔐 1️⃣ Confidentialité

## Définition

Empêcher l’accès non autorisé aux données

---

## Exemples

| Cas | Description |
|---|---|
| mot de passe | protection |
| chiffrement | TLS |
| accès restreint | permissions |

---

## Moyens

| Solution | Description |
|---|---|
| chiffrement | données |
| authentification | login |
| contrôle accès | IAM |

---

# 🧩 2️⃣ Intégrité

## Définition

Garantir que les données ne sont pas modifiées

---

## Exemples

| Cas | Description |
|---|---|
| fichier altéré | attaque |
| hash | vérification |
| signature | validation |

---

## Moyens

| Solution | Description |
|---|---|
| hash | SHA |
| signature | numérique |
| contrôle | logs |

---

# ⚡ 3️⃣ Disponibilité

## Définition

Garantir l’accès aux données et services

---

## Exemples

| Cas | Description |
|---|---|
| serveur down | panne |
| DDoS | attaque |
| crash | indisponible |

---

## Moyens

| Solution | Description |
|---|---|
| redondance | HA |
| sauvegardes | backup |
| monitoring | supervision |

---

# 📊 Résumé CIA

| Élément | Objectif | Exemple |
|---|---|---|
| Confidentialité | protéger | chiffrement |
| Intégrité | fiabilité | hash |
| Disponibilité | accès | HA |

|---|---|

# 🎯 Conclusion

La triade CIA est :

- la base de toute stratégie sécurité
- utilisée dans toutes les normes (ISO, NIST)

👉 Toute mesure de sécurité doit répondre à :

- protéger
- garantir
- assurer
