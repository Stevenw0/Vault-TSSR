## Summary

Cette source rapporte que Microsoft aurait lance un projet interne appele Windows K2 pour ameliorer les performances, reduire certains irritants de Windows et rattraper le retard percu face a Linux, SteamOS et Proton dans le jeu video.

## Key Claims

- D'apres l'article, Windows K2 aurait commence fin 2025 et viserait un changement de culture interne autour du developpement Windows.
- Le projet mettrait l'accent sur les performances, l'interface, la reduction de fonctions jugees trop presentes et la modernisation de vieux composants.
- Microsoft viserait un rattrapage de SteamOS en performance jeu sur un horizon d'un a deux ans.
- Windows Update serait aussi vise, avec l'objectif rapporte de limiter les redemarrages a une fois par mois.
- Les premiers changements importants seraient attendus en preview a partir de l'ete 2026, sans sortie unique du projet.
