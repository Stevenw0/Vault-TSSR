# Parcours Windows Server et Active Directory

## Objectif

Comprendre Windows Server, Active Directory, les strategies de groupe, le deploiement, la reparation et l'administration distante.

## Ordre conseille

## 1. Poser les bases Windows Server

1. [Windows Server 2022](/pages/notes/windows-server-2022.md) - Decouvrir la plateforme serveur.
2. [Client Access License](/pages/notes/client-access-license.md) - Comprendre les licences CAL.
3. [Bottleneck](/pages/notes/bottleneck.md) - Identifier les goulots d'etranglement.
4. [Windows 12](/pages/notes/windows-12.md) - Lire comme culture d'evolution Windows.
5. [Projet Windows K2](/pages/notes/projet-windows-k2.md) - Suivre les pistes rapportees d'amelioration Windows autour des performances et de l'UX.
6. [Windows Server 2025](/pages/notes/windows-server-2025.md) - Situer la version 2025, ses nouveautes et ses retraits.
7. [Windows Server 2025 - nouveautés et impacts](/pages/notes/windows-server-2025-nouveautes-et-impacts.md) - Preparer les impacts de migration.

## 2. Comprendre Active Directory

5. [Active Directory](/pages/notes/active-directory-ad.md) - Comprendre domaine, DC, objets et structure logique.
6. [AD Complet](/pages/notes/ad-complet.md) - Approfondir avec commandes et scenarios.
7. [Kerberos](/pages/notes/kerberos.md) - Comprendre l'authentification AD.
8. [Security Account Manager](/pages/notes/security-account-manager.md) - Situer SAM et authentification locale.
9. [SYSVOL AD](/pages/notes/sysvol-ad.md) - Comprendre le partage critique des GPO.
10. [AGDLP](/pages/notes/agdlp.md) - Structurer les permissions avec groupes globaux et locaux.
11. [SoD](/pages/notes/sod.md) - Séparer les rôles critiques et les responsabilités.

## 3. Administrer les strategies

12. [GPO](/pages/notes/gpo.md) - Comprendre GPC/GPT et ordre d'application.
13. [Fichiers ADMX](/pages/notes/fichiers-admx.md) - Comprendre les modeles d'administration.
14. [LGPO](/pages/notes/lgpo.md) - Automatiser les strategies locales.
15. [Windows LAPS](/pages/notes/windows-laps.md) - Gerer les mots de passe admin locaux.
16. [ADCS](/pages/notes/adcs.md) - Ajouter la brique certificats.

## 4. Deployer, reparer et imager Windows

17. [WinPE](/pages/notes/winpe.md) - Comprendre l'environnement de preinstallation.
18. [Sysprep](/pages/notes/sysprep.md) - Generaliser une image.
19. [PXE](/pages/notes/pxe.md) - Comprendre la chaine de boot reseau.
20. [WDS](/pages/notes/wds.md) - Deployer par reseau.
21. [MDT - Upgrade Windows 10 vers Windows 11](/pages/notes/mdt-upgrade-windows-10-vers-windows-11-tutoriel.md) - Automatiser une sequence de migration.
22. [FOG Project](/pages/notes/fog-project.md) - Comparer avec une solution de clonage.
23. [FOG Project - Lab VMware](/pages/notes/fog-project-lab-vmware.md) - Lire un lab valide de capture et de deploiement FOG sur VMware.
24. [DISM](/pages/notes/dism.md) - Reparer et maintenir des images.
25. [DISM - reparer Windows](/pages/notes/dism-reparer-les-fichiers-systemes-et-images-de-windows-11-10.md) - Approfondir les usages de reparation.
26. [Boot Configuration Data](/pages/notes/boot-configuration-data.md) - Comprendre le demarrage Windows.

## 5. Administrer a distance et maintenir

27. [WinRM](/pages/notes/winrm.md) - Comprendre l'administration distante moderne.
28. [WMI](/pages/notes/wmi.md) - Interroger et administrer Windows.
29. [RDP](/pages/notes/rdp-remote-desktop-protocol.md) - Connaitre le bureau a distance.
30. [RPC](/pages/notes/rpc.md) - Situer les appels distants.
31. [WSUS](/pages/notes/wsus.md) - Gerer les mises a jour.

## 6. Stabiliser l'environnement domaine

32. [NTP et PTP](/pages/notes/ntp-et-ptp.md) - Comprendre la synchronisation temps.
33. [w32tm](/pages/notes/w32tm.md) - Administrer le temps Windows/domaine.

## 7. Projet integre

34. [Documentation Lab](/pages/notes/documentation-lab.md) - Suivre le deploiement SafeGuard avec domaine `safeguard.lan`, DC1, DC2, SMB, AGDLP, imprimante et GPO.
35. [Resume Lab](/pages/notes/resume-lab.md) - Reviser l'architecture et les validations Windows du lab.
36. [Projet SafeGuard](/pages/notes/projet-safeguard.md) - Lire la synthese maintenue du projet.

## Validation

Tu sais que ce parcours est acquis si tu peux expliquer comment installer un serveur, creer un domaine, appliquer des GPO, authentifier un utilisateur, deployer une machine, publier des ressources et administrer le tout a distance.
